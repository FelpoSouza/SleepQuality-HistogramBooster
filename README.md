# SleepQuality-HistogramBooster
Classificador pautado em um dataset de qualidade do sono obtido no Kaggle; utilizando do "Histogram Gradient Boosting Classifier" do sklearn.
O dataset especificado em si se encontra no link < https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset >.
Verifique se o arquivo "Sleep_health_and_lifestyle_dataset.csv" está no mesmo diretório que "Classsificador.py" e "Graficos.py".
